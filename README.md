## Tech Stack

**Languages:** ReactJS

## Run Locally

Clone the project

```bash
  git clone https://github.com/hodanmohamed/portfolioapp.git
```

Go to the project directory

```bash
  cd portfolioapp
```

Install dependencies

```bash
  npm install
```

Start the server
